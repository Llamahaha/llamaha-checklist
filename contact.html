Before You Reach Out

The fastest way to get a useful response

Include the basics

Share the application or page involved, what you were trying to do, what changed, and the exact message or symptom you saw.

Send the details that narrow the issue

  • The exact work account you used, especially if the app can also use a personal account.
  • The app name and version if the app opens far enough to show it.
  • One screenshot of the exact message, missing screen, or failed step.
  • Whether the same task works in the browser, if the app has a browser version.
  • Whether the same task works from another device, Cloud PC, Citrix session, or VPN path if one is available.
  • The exact file, project, mailbox, printer, site URL, shared path, datasource, company file, or phone prompt involved.

Point to the source

If you are reporting a content issue, include the guide URL and the section that looks missing, incorrect, or unclear.

Use the right path

For immediate help, use Search, PC Help, App Help, Licensing Help, or Tips & Tricks. For page corrections, missing coverage, or help-center questions, email the admin address below.
