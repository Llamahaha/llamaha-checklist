Privacy Note

Keep customer-specific data out of the public site

The site content is designed to stay generic and reusable. The known issue tracker saves only in the local browser on the current device, and it should still be used only for sanitized issue patterns and non-sensitive workaround notes unless you add sign-in and secured storage later.
